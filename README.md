## About me

I'm an **Applied AI Engineer** and **Full Stack Developer** with 4+ years of experience building practical AI systems with **LLMs, RAG, agents, APIs, automation workflows and data-driven applications**.

My work combines software engineering, generative AI, backend development, data systems and product thinking to transform manual processes into scalable, useful and measurable solutions.

Currently, I work with **Python, FastAPI, LangChain, LangGraph, PostgreSQL/pgvector, Docker, Angular, Next.js, APIs and LLM providers such as OpenAI, Claude and Gemini**.

I build systems that connect AI with real business operations: **CRM automation, internal tools, dashboards, RAG pipelines, WhatsApp bots, API integrations and decision-support applications**.

I also work with **Prompt Engineering, Embeddings, Chunking, Langfuse, AI/LLM Observability, MCP, Spec-Driven Development, GitHub Spec Kit, Token Optimization and Cost Optimization** to improve reliability, traceability and efficiency in AI applications.
